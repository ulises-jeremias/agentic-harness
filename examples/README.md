# Examples

> Three minimal walkthroughs for common AI Workspace setups.

| Example | What it shows |
|---------|--------------|
| [solo-dev-daily-triage](solo-dev-daily-triage/) | Solo developer running a daily issue triage loop (L1) |
| [pr-babysitter](pr-babysitter/) | Two-repo PR review loop with workspaced context (L2) |
| [oss-contribution](oss-contribution/) | OSS contribution workflow with knowledge-driven discovery |

Each example is self-contained: a `setup.sh` you can run, a sample pack YAML,
and a loop YAML ready to `bin/loop init` from.
