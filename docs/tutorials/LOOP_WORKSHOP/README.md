# Loop Creation Workshop — agentic-harness

> Progressive tutorial series: build autonomous AI processes with safety tiers.

**Time to complete**: ~2 hours total (30 min per part + exercises)
**Prerequisites**: Completed ["Your First PR" tutorial](../FIRST_PR.md), harness installed

---

## What You'll Build

By the end of this workshop, you'll have three working autonomous loops:

| Part | Loop | Tier | What it does |
|------|------|------|-------------|
| 1 | Daily Issue Triage | L1 | Scans open issues, writes a report — read-only |
| 2 | PR Babysitter | L2 | Reviews open PRs, posts draft comments — human approval gate |
| 3 | CI Sweeper | L3 | Detects CI failures, opens fix PRs — autonomous |
| 4 | Capstone | Custom | Design your own loop using the loop engineering discipline |

Each part produces a working loop you can run immediately. Parts build on each other — you can stop at any tier.
